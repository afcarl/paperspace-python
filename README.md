Paperspace API for Python
=========================

Sample usage:

    import paperspace

    paperspace.config.PAPERSPACE_API_KEY = '14a4bc1cbc414...'

    paperspace.jobs.runas_job({'project': 'myproject', 'machineType': 'GPU+', 'container': 'Test-Container'})

    print('hello paperspace!')
